# ASK BuildEase - Making Construction Seamless

A modern, responsive digital marketplace built with React for construction materials, services, and solutions in India.

## About ASK BuildEase

ASK BuildEase is revolutionizing the Indian construction industry by providing a comprehensive digital marketplace that connects small and medium construction firms, real estate developers, contractors, and builders with trusted suppliers.

## Features

- **Home Page**: Browse construction materials with search and category filtering
- **Shopping Cart**: Add, remove, and update product quantities
- **Checkout**: Complete order form with validation
- **About Page**: Learn about our mission, offerings, and value proposition
- **Contact Page**: Get in touch with our team

## Tech Stack

- React 18
- React Router DOM
- Context API for state management
- Modern CSS with responsive design

## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Building for Production

```bash
npm run build
```

## Project Structure

```
src/
  ├── components/       # Reusable components
  ├── context/         # Context providers (Cart, Products)
  ├── pages/           # Page components
  └── App.js           # Main app component with routing
```
